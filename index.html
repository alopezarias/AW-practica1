Instrucciones del pasatiempo

Identifique la primera palabra y la última de cada bloque con las pistas que se dan. A continuación, trate de descrubir las palabras intermedias. Para lograrlo, cambie una letra de la primera palabra para obtener la próxima y, después, altere el orden de una o varias letras para encontrar la siguiente. Siga así, sucesivamente, hasta que logre completar ambos casilleros. Todas las palabras intermedias deben tener significado.

Pistas de resolución

Introduce letras y obtendrás palabras formadas con ellas. Úsame con precaución, solo tienes 3 oportunidades.
